📚 PulseEvents Documentation

Full guide for installation, commands and configuration

📦 Overview

PulseEvents is a Minecraft plugin that automatically manages server events. It handles event selection, announcements, countdowns and execution using a BossBar system.

⚙️ Installation

1. Download the plugin JAR file

2. Place it into your plugins/ folder

3. Restart your server

4. Configure files in /plugins/PulseEvents/

🧭 Commands

/pe help – Show all commands
/pe start – Start a random event
/pe stop – Stop current event
/pe status – Show current event
/pe reload – Reload config & language
/pe update – Check for updates

🌐 Permissions

pulseevents.admin – access to all commands
pulseevents.use – basic usage permissions

⚡ Features

  • Automatic event system
  • BossBar countdown
  • Random event selection
  • Multi-world support
  • Language system (EN/PL)

⚠️ Notes

  • Events must be pre-registered in config
  • Players cannot create events in-game
  • Plugin is still under active development
