📦 PulseEvents

PulseEvents is a Minecraft plugin that automatically runs server events with announcements and a BossBar countdown system.

Events are randomly selected from a predefined pool and managed automatically without requiring manual player setup.

⚡ Features

🔹 Automatic event announcements
🔹 BossBar countdown system
🔹 Random event selection
🔹 Auto start / stop system
🔹 Multi-world support
🔹 Permission-based commands
🔹 Anti-spam cooldown system

🧭 Commands

/pe help – show commands
/pe start – start random event
/pe stop – stop event
/pe status – check status
/pe reload – reload config
/pe update – check updates

🌐 Languages

✔ English (EN)
✔ Polish (PL)

⚠ Behavior

Events are automatically announced.
BossBar countdown is displayed before start.
Events run for a defined duration and stop automatically.
